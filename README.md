# Real Mart Sales Analysis Dashboard | Power BI

## Overview

An interactive Power BI dashboard built to analyze Real Mart sales data and generate business insights using data visualization techniques.

## Features

* KPI Dashboard
* Sales Trend Analysis
* Category-wise Sales Analysis
* Geo-location Visualization
* Payment Mode Insights
* Interactive Filters and Slicers

## Tools Used

* Power BI
* DAX
* Power Query
* Data Visualization
